# minitwit

# Getting Started

## Prerequisites

* npm
* Git

## Installation

1. Clone the repo
   ```
   git clone 
   ```

2. Install NPM packages
   ```
   npm install
   ```

3. Config .env file
   ```
   There is an exemple of .env file named .env.example in /apps/api and /apps/frontend
   ```

## Run apps

1. API
   ```
   npm run dev -w apps/api
   ```

2. Frontend
   ```
   npm run dev -w apps/api
   ```

# Run tests

```
npm run test -w apps/api
npm run test -w apps/frontend
```
